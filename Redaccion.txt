El visualizador de simulaciones y modelos, abreviado como VySM, es una implementación que busca resolver problemas 
de interpretación de resultados en experimentos. Uno de los puntos mas fuertes de este simulador es la utilización de 
tecnologias Web actuales como Javascript, HTML y CSS3. Hoy en día la mayoria de los sitios web´s se encuentran codificados con 
estas tecnologias, asi de esta manera, se quiso aprovechar este potencial y crear un simulador que permitiera al sector de la 
investigacion visualizar sus resultados sin necesidad de instalar ningún programa adicional. El simulador corre detro del 
navegador web y tiene la posibilidad de usar la tarjeta gráfica del computador si asi se ocupase, las visualizaciones se 
logran gracias a una libreria de Javascript llamada three.js, esta libreria permite crear animaciones de acuerdo a nuestras 
necesidades. El simulador VySM ocupa archivos en formato JSON, dentro de estos archivos se encuentra declarado el código necesario
para poder visualizar los modelos deseados. Por ahora el visualizador muestra diagramas de Voronio, digramas de particulas y 
diagramas porosos. Mas adelante se explicará el funcionamientoo de estos archivos JSON asi como su codificacion en Javascript.
